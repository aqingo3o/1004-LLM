## Repo Structure
資料夾的結構是，大分類的資料夾(tutorial, interface...）之下，有以**人類**命名的 subfolder  
詳見 [CONTRIBUTING.md](CONTRIBUTING.md)  
要重組自己的檔案結構建議建立 banch，要改**別人**的檔案結構建議先推 issue  
以下是地圖

### REDAME.md  
如同 description 所述, 這是一個實驗室共用的 repo,  
為了避免您迷路，請先閱讀 `README.md` to understand what this place is and how things are organized.  

### CONTRIBUTING.md
除了炒炒同學也請炒點有用的東西!  
`CONTRIBUTING.md` 中描述了一些格式或結構的規範。在推東西上來之前請先閱讀它，避免爆破這個 repo。

### start-with-blackhole.md
Simple commands on how to remotely connect to the server(named **blackhole**).  
Ypu can and should give them a try.

### whiteBoard.md
可以在這邊留言ㄡ

### exp/
實驗性質的東西都可以放在這裡  
請依照 [CONTRIBUTING.md](CONTRIBUTING.md) 新增您的個人 sub-folder

### tutorial/
放一些你覺得是**面對人類**的教程，我們感謝您的付出  
關於教程內的任何問題可以新增 issue  

### interface/
如果有寫聊天介面的話可以放在這邊  
輕再買 javascript 書的時候不要

## 旗魚

有普通的建議可以建立分支或推 issue

## Repo Structure
以下是地圖

### REDAME.md  
如同 description 所述, 這是一個實驗室共用的 repo,  
為了避免您迷路，請先閱讀 [README.md](README.md) to understand what this place is and how things are organized.  

### CONTRIBUTING.md
除了炒炒同學也請炒點有用的東西!  
[CONTRIBUTING.md](CONTRIBUTING.md) 中描述了一些格式或結構的規範。在推東西上來之前請先閱讀它，避免爆破這個 repo.  
資料夾的結構是，大分類的資料夾(`Tips/`, `tutorial/`, `interface/`...）之下，有以**人類**命名的 subfolder.  
要重組自己的檔案結構建議建立 banch，避免多出很多無用 commit,  
要改**別人**的檔案結構建議先推 issue, 禮貌一點總是好的...?

### exp/ 
This should be the most frequently used folder. Feel free to put anything experimental here, it is essentially 一鍋雜燴.  
Please add your personal sub-folder according to [CONTRIBUTING.md](CONTRIBUTING.md).

### tutorial/
放一些你覺得是**面對人類**的 LLM-related 教程，我們感謝您的付出。  
關於教程內的任何問題可以新增 issue.
